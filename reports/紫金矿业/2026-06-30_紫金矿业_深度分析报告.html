一、商业模式深度分析

1.1 公司如何赚钱

紫金的盈利本质是一个"低成本采矿 → 高价值金属 → 全球销售"的资源转化生意。

公司在全球范围内勘探、收购、建设并运营矿山，将地下的铜矿石、金矿石、锂矿石挖出来，经过选矿→冶炼→精炼加工后，以国际市场价格出售金属产品。

核心逻辑：资源储量大 + 采矿成本低 + 金属价格长期偏高 = 超额利润。

1.2 核心产品与服务

产品 2025年产量 毛利率 说明
矿产铜 109万吨 ~62% 收入压舱石，全球第4
矿产金 90吨 ~65% 高毛利，增速加快
矿产锂 2.5万吨LCE ~28% 爬坡期，未来爆发力最大
矿产锌/铅 45万吨 ~45% 稳定现金流来源
矿产银 437吨 ~50% 铜金矿副产品

1.3 收入来源结构

2025年总营收3,491亿元，结构如下：

  • 铜业务约52%（~1,815亿）——公司命脉，铜价每涨500元/吨即增净利润约5亿+
  • 金业务约35%（~1,222亿）——2025年金价历史高位显著拉动
  • 锂业务约3%（~105亿）——从零到量产，2026年目标占比将快速提升
  • 其他（锌银钼铁）约10%——稳定现金牛

1.4 客户是谁

紫金是典型的B2B大宗商品企业，不面对终端消费者：

  • 冶炼厂/精铜加工商：紫金的铜精矿→全球铜冶炼企业
  • 黄金精炼商/央行：金锭/金精矿→直接进入国际金价体系
  • 新能源产业链：碳酸锂→电池厂（宁德、LG等）
  • 国际贸易商：嘉能可等国际大宗交易商

客户分散在全球43个国家，单一客户依赖度极低，无议价风险。

1.5 行业位置

紫金处于全球矿业食物链的最顶端——一次性资源开采型企业，是铜铁矿工业和新能源产业的上游原材料提供者。

其独特之处在于：中国资本+全球矿权+中国工程执行力，构成了一种其他矿业巨头难以复制的竞争模式。

1.6 竞争对手

公司 优势 劣势（对比紫金）
Codelco（智利） 全球最大铜企 国有体制，扩张灵活性差

紫金的差异化：唯一一家同时在铜（全球前5）、金（全球前6）、锂（全球前5储量）三个赛道同步领先的公司。

1.7 核心竞争优势

  1. 低成本工程能力：中国工程师能在西藏5,000米海拔建矿，成本比外资方案省30%。这是技术+执行力的护城河。
  2. 资源储量极深：铜1.1亿吨、金4,000吨、碳酸锂1,800万吨（2024年末），支撑50年以上开采。
  3. 增速全球第一：铜产量近5年CAGR 24%，是所有主流矿企中增速最快的。
  4. 成本曲线左侧：大多数主力矿山生产成本低于全球平均，抗周期能力强。
  5. 多金属协同：铜矿中伴生金/银/钼，等于同一座矿山同时贡献多个收入来源，边际成本趋零。

1.8 网络效应、规模效应、品牌优势

  • 规模效应：产量越大，单位固定成本越低。卡莫阿铜矿从30万吨扩到60万吨，单位成本可以继续下降20%+。
  • 品牌议价力（有限但存在）：在全球铜精矿贸易中，紫金已是"稳定可靠供应商"。
  • 政策资源壁垒："央企+地方国资+民营"混合股权结构，能拿到国内最优质的矿权（如西藏巨龙）。这是一种制度性护城河。

1.9 未来5-10年增长驱动力

  1. 铜——能源转型超级周期：每辆电动车用铜约60kg，是燃油车的4倍；光伏、储能、特高压电网均是铜的消耗大户。全球铜供给缺口持续扩大，紫金正是全球增量最大来源。
  2. 金——"去美元化"叙事长期化：央行连续净购金，地缘政治不稳，黄金中枢上移。海域金矿（达产后年产15-20吨）等重大项目将在2026年后陆续放量。
  3. 锂——第三增长极爆发：2025年~2.5万吨→2028年27-32万吨LCE，增幅超10倍。盐湖提锂成本优势显著（3.5-4万元/吨）。
  4. 并购加速：联合黄金280亿收购若完成，一次性新增500吨黄金资源量、年产潜力25吨。
  5. 巨型项目产能爬坡：卡莫阿三期（60万吨）、巨龙二期（35万吨）、塞尔维亚改扩建（45万吨）、朱诺铜矿（7.6万吨）四个超级项目2025-2028年陆续达产。

1.10 总结：为什么紫金矿业可能成为未来十年的赢家？

  1. 押注的赛道是对的：铜（电气化）、金（货币体系重构）、锂（新能源）是未来10年最确定的三条长期需求主线，紫金在这三条赛道均处于全球领导地位。
  2. 竞争优势不易被复制：中国工程团队在极端地质条件下的低成本建矿能力，是紫金独有的护城河。
  3. 增长还远没到顶：2025年净利518亿，目前多个世界级项目仍在爬坡，乐观情景下2028年净利润有望突破1,000亿。

风险提示：大宗商品价格波动、海外政治风险（非洲/南美）、锂价若长期低位则第三极逻辑弱化。

二、行业研究

2.1 行业规模与TAM

三大矿种2025年合计TAM约7.9万亿元人民币：

品种 全球市场规模 备注
铜矿 ~2.6万亿元 精炼铜~2,900万吨 × $9,000/吨
黄金 ~5.0万亿元 矿产+再生约5,030吨 × $3,000/盎司均价
锂矿 ~0.3万亿元 碳酸锂~120万吨LCE × 7万/吨

TAM核心特征：不受总量壁垒约束——需求由实体经济硬性驱动，只受金属价格周期影响振幅。

2.2 行业增长率

  • 铜：全球需求CAGR（2025-2030）2.3-2.9%，其中新能源车用铜CAGR 14%、储能用铜CAGR >50%。供给增速仅0.9-1.4%，持续跑输需求。
  • 黄金：需求CAGR（2024-2030）约3.2%，金价中枢已永久上移至$3,000+，央行连续5年净购金。
  • 锂：全球锂需求CAGR（2025-2030）高达35-50%，但当前处于产能严重过剩的价格低谷期。

2.3 行业生命周期

细分 生命周期阶段 核心逻辑
传统铜矿（地产/家电） 成熟期 中国地产见顶，传统需求趋于稳态
能源转型铜（新能源/电网/AI） 高速成长期 结构性新需求，取代旧动能
黄金 成熟高景气期 价值中枢上移，去美元化叙事长期化
锂矿 成长早期（周期低谷） 供需磨底，2027年后第二轮景气周期

铜矿是非常罕见的 "成熟期行业+新成长曲线叠加"——类比石油在1900年代遇到汽车普及，是百年一遇的需求重塑。

2.4 行业壁垒（评级：极高）

  1. 资源稀缺性：全球铜矿品位25年跌去60%（1.34%→0.52%），新发现世界级铜矿越来越难
  2. 超长建设周期：勘探→审批→建设→达产平均7-10年。供给弹性极差
  3. 地缘政策护城河：全球45%铜矿在智利/秘鲁，原住民抗议、环保审批是系统性壁垒
  4. 工程技术壁垒：低品位矿生物冶金技术、高海拔建矿能力，20年经验沉淀
  5. 资本门槛：一座世界级铜矿前期资本开支50-200亿美元
  6. 一体化供应链：精矿→冶炼→精炼→贸易一体化

2.5 行业未来十年趋势

  1. 能源转型：电动车（单车用铜80-130kg）、风光储能、特高压电网
  2. AI基础设施：数据中心、AI服务器大量用铜，2025-2030年新增用铜206万吨
  3. 去美元化浪潮：央行持续增持黄金，价格中枢抬升
  4. 锂能源革命：锂电池仍是主流，2030年全球碳酸锂需求有望突破200万吨LCE

2.6 三情景分析

最乐观情景（概率30%）

  • 全球能源转型全面提速，IEA激进情景兑现
  • 电动车全球渗透率2030年突破50%
  • AI算力军备竞赛持续
  • 主要矿区频繁扰动，供给低于预期
  • 结果：2030年铜供需缺口>200万吨，铜价突破$15,000-20,000/吨，金价$3,500+，锂价12-15万/吨

基准情景（概率50%）

  • 能源转型按IEA基准情景推进
  • 全球经济温和增长（GDP年均2-3%）
  • 新矿资本开支缓慢回升，但无法填补缺口
  • 结果：2030年缺口145-184万吨，铜价$11,000-14,000/吨，金价$3,200-3,800/吨，锂价10-12万/吨

最悲观情景（概率20%）

  • 全球经济深度衰退（类2008）
  • 废铜回收技术革命性突破
  • 新矿集中投产，超预期供给
  • 固态电池/钠电池提前突破
  • 结果：铜价跌回$7,000-9,000/吨，金价回落至$2,200-2,500/吨，锂矿持续低迷5年以上

2.7 最终判断：行业是否具备长期成长性？

研究员评级：强烈确认，属于大宗商品中少见的"结构性成长"赛道

  1. 供给侧的刚性约束是客观物理规律，无法被政策或资本短期克服
  2. 需求侧的新驱动是文明级别的转型，生命周期30-50年
  3. 金价价值中枢上移有货币体系背书
  4. 铜金锂矿业是未来十年兼具"周期弹性+成长确定性"的稀缺赛道

逐项详情

3.1 市场潜力：9分

铜：能源转型+AI数据中心→需求CAGR 2.3-2.9%，供给增速仅0.9-1.4%。黄金：去美元化+央行购金→金价中枢永续上移。锂：新能源车渗透率冲击50%→需求CAGR 35-50%。

3.2 管理层能力：8分

陈景河32年从闽西县矿做到全球前3矿企。三大经典战役：2015年底部$2.98亿抄底卡莫阿、2018年入手塞尔维亚蒂莫克、2020年¥38.8亿拿下巨龙。股权激励行权条件绑定营收+ROE+负债率>行业75分位。扣分项：陈景河2025年底卸任，新掌门未经完整周期。

3.3 研发能力：8分

"矿石流五环归一"：地质勘查→采矿→选矿→冶金→环保并联协同，吨矿综合成本较行业低20-25%。低品位矿开发独门绝技（紫金山金矿入选品位仅0.6g/t仍盈利）。勘探成本仅$11.7/盎司（行业$32.3）。

3.4 销售能力：6分

产品是标准化大宗商品（铜精矿、金锭、碳酸锂），价格由LME/LBMA全球定价。B2B大客户模式，不依赖终端营销。矿业公司销售能力可适度下调权重。

3.5 利润率改善空间：8分

矿产铜毛利率~62%，矿产金~65%。改善来自：(1)巨型项目爬坡降本（单位固定成本再降15-20%）；(2)伴生金属协同（边际成本趋零）；(3)锂矿从负→正利润拐点；(4)规模效应持续释放。

3.6 护城河：9分

六维自增强闭环：技术（五环归一）→资源（铜1.1亿吨/金4000吨/锂1800万吨LCE）→成本（全球最低20%区间）→效率（"紫金速度"建设短30%投资低25%）→财务（ROE 22%+）→战略（逆周期并购），环环相扣，拆掉任何一环都需同时攻破其他环节。

2025年ROE 33.04%，ROIC估算约16-18%，全球矿企平均8-10%。持续且显著高于WACC，存在真正的经济护城河。

4.11 会计政策：🟡

5.1 六大结构性裂缝

🔴 裂缝一：卡莫阿矿震 — "皇冠上的明珠"已经碎裂

紫金的"铜产量增速全球第一"叙事，扣除卡莫阿后净增量可能仅为个位数。

🔴 裂缝二：锂矿 — 增长故事最大的"皇帝新衣"

  • 碳酸锂从2022年60万元/吨跌至2025年跌破6万元/吨（跌去90%）
  • 高盛预测2026H2-2028年锂市场20-22%供给过剩
  • 阿根廷3Q盐湖完全成本约4.5-5万元/吨，在6万/吨价格下毛利率仅10-20%
  • 2026年计划12万吨LCE的实际利润可能不到市场预期的1/5

🟠 裂缝三：280亿联合黄金并购 — 周期顶部的自杀式豪赌

四个需要质询的问题： 1. 为什么在金价$3,000+历史高位进行10年来最大并购？ 2. 为什么要进入全球最危险的马里（萨赫勒战区，使馆正在撤侨）？ 3. 为什么收购一家2023-2024年连续亏损的公司？ 4. 280亿+卡莫阿复产支出，资产负债率将被推回55%以上？
